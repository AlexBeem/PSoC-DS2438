Detailed Description

DS2438 Library with 1-Wire interface.

This library allows to use the DS2438 device on PSoC 3/5LP devices. The current implementation only allows to have one single DS2438 in your system, as all the functions perform a skip ROM command when sending commands to the device.

uint8_t DS2438_WriteOffset ( int16_t  offset)

Write new offset value.

This function writes a new offset value so that small currents are removed from current values. Because small current ADC offset errors can have a large cumulative effect when current is integrated over time, the DS2438 provides a method for canceling offset errors in the current ADC. After each current measurement is completed, the measured value is added to the contents of the Offset Register and the result is then stored in the Current Register. The Offset Register is a two-byte nonvolatile read/write register formatted in two’s-complement format.

Parameters
offsetnew offset value to be written.
Return values
#DS2438_OKif device is present on the bus.
#DS2438_DEV_NOT_FOUNDif device is not present on the bus.
